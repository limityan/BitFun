§02 · 先把 "Harness" 这个词讲清楚

Harness · Engineering

Etymology · 词源 / 直译
harness马具 · 挽具
/ˈhɑːnɪs/ · 名词 · 动词

"directs a powerful animal's energy toward useful work without letting it run wild"

把强大动物的能量导向有用工作，
同时不让它跑偏。

—— Anthropic 在 Effective Agents 文档中
把这个词正式引入 Agent 工程语境。

The Working Formula · 工作公式
Agent =
Model + Harness
↑ Model 决定上限 capability ceiling
↓ Harness 决定下限 reliability floor

公式由 LangChain 在 "Anatomy of an Agent Harness" 中提出，
已被 Anthropic / OpenAI 文档广泛沿用。
