Three Harness Changes · 三处 Harness 改造
Harness · 评估 LangSmith Trace + Trace Analyzer Skill
每个 agent action 全 trace 入库；自研 Trace Analyzer Skill 三步：拉 trace → 并行 sub-agent 分析 → 主 agent 综合改 prompt。不靠人工调 prompt，靠 trace 调 prompt。
Harness · 编排 Build-Verify Loop · 4 阶段
系统提示词强制 4 阶段：Plan & Discovery → Build → Verify → Fix。Verify 阶段必须 对照 task spec 而非 agent 自己的代码。PreCompletionChecklistMiddleware 在 exit 前 hook 一次强制再 verify（"Ralph Wiggum Loop"）。
Harness · 记忆 + 编排 Context Injection + Loop Detection
  • 环境上下文主动注入：sandbox / 工具 / runtime 约束在开局即给，避免试错
  • LoopDetectionMiddleware：per-file edit count，超过 N 次注入 "consider reconsidering"
  • Adaptive Reasoning：每个子任务动态选 high / xhigh，避免 token 浪费
同一模型，仅优化 Agent Harness，排名从 30+ 提升到 Top5。
