§02 · 与 Prompt / Context Engineering 的关系

三个概念是包含关系，不是并列。

Harness Engineering
Context Engineering
Prompt Engineering
Layer 1 · innermost
Prompt Engineering
研究如何向模型说话 —— 角色设定、指令格式、输出约束。
Layer 2 · middle
Context Engineering
研究如何把信息正确组装进上下文 —— 召回、压缩、注入、窗口管理。
Layer 3 · outermost
Harness Engineering
研究整个工作过程如何不出事 —— 涵盖以上两者，加上工具、循环、记忆、反馈、约束、评估。
三者非对立。Harness 是更外层的"壳"，子集是另两者。
Prompt  ⊂  Context  ⊂  Harness
