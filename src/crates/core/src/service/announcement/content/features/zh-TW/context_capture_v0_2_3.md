# 聊天截圖與短錄屏

BitFun 現在支持在桌面端直接捕獲 **單張截圖** 或 **短錄屏**，並把結果直接附加到當前聊天草稿中。

當前版本支持：
- 桌面端 **截圖捕獲**
- 最長 **10 秒** 的短錄屏
- 錄屏會保存為 **一個受管的短視頻**
- 發送給多模態模型前，會自動從視頻中抽取少量幀進行分析

<!-- page -->

# 如何使用

你可以在聊天輸入框中使用新的入口：
- **截圖**：立即捕獲當前屏幕並附加 1 張圖片
- **開始錄屏**：啟動一次短時錄屏
- **停止錄屏**：提前結束；如果不手動停止，BitFun 會在 **10 秒** 後自動結束

在附加到消息前，BitFun 會先在本地完成壓縮和降采樣，這樣可以控制存儲體積，也能降低多模態模型的額外成本。受管視頻會保留在本地會話 artifacts 中用於預覽和歷史恢復；發送給模型時使用抽取幀和元數據。

# 隱私提醒

截圖和錄屏裡可能包含：
- 密碼、驗證碼
- 通知、私聊消息
- 客戶數據或內部業務信息

附加截圖或錄屏前，BitFun 會先要求隱私確認。只有勾選 **不再提示** 並繼續後，才會記住此選擇。

你始終保有控制權：
- 不會自動發送
- 不會在後台偷偷錄屏
- 不會錄製音頻
- 發送前可以移除任意截圖或錄屏
